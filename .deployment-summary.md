## Quick Commands

### Switch Between Branches
```bash
# View current version
git checkout main

# View upgraded version
git checkout marketing-upgrade
```

### Deploy Updates
```bash
# Deploy current version to production
git checkout main
vercel deploy --prod

# Deploy upgraded version to preview
git checkout marketing-upgrade
vercel deploy --yes
```

### Live Development
```bash
# Start local dev server (works on both branches)
bun run dev
```
