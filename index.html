Quick Start Guide

Get up and running in minutes

Install VIVIM SDK using npm or yarn:

npm install @vivim/sdk
yarn add @vivim/sdk

Initialize VIVIM with your API key:

import { VivimClient } from '@vivim/sdk';

const vivim = new VivimClient({
  apiKey: process.env.VIVIM_API_KEY,
  encryption: true
});

Store and retrieve memories:

// Store a memory
await vivim.memory.store({
  type: 'conversation',
  content: 'Discussion about project architecture',
  tags: ['architecture', 'project']
});

// Retrieve memories
const memories = await vivim.memory.search({
  query: 'architecture',
  limit: 10
});

Create and share context recipes:

// Create a context recipe
const recipe = await vivim.context.createRecipe({
  name: 'Dev Sprint',
  memories: ['mem_123', 'mem_456'],
  shareWith: 'community'
});

// Share the recipe
await vivim.context.share(recipe.id);

Manage your privacy settings:

// Set privacy level
await vivim.privacy.setLevel('private');

// Configure encryption
await vivim.privacy.configureEncryption({
  algorithm: 'AES-256-GCM',
  keyRotation: 'monthly'
});

Sync your knowledge graph across devices:

// Enable P2P sync
await vivim.sync.enable({
  peerDiscovery: true,
  autoSync: true,
  bandwidth: 'high'
});

// Manual sync
await vivim.sync.syncNow();
