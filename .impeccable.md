## Design Context

### Users
- **Mixed audience**: Landing page appealing to both AI developers building memory infrastructure AND individual power users who use ChatGPT/Claude daily
- Primary context: Landing page to drive sign-ups and demo exploration
- Users want: Sovereignty over their AI data, portability across providers, tools to manage/engage with any AI

### Brand Personality
- **Voice**: Anti-establishment, user-empowering, "flipping the script" on Big AI
- **Tone**: Confident, innovative, privacy-forward but not preachy
- **3-word personality**: Sovereign • Innovative • Unapologetic
- **Emotional goal**: Make users feel they finally have control over their AI memory — excitement + empowerment, not fear

### Aesthetic Direction
- **Visual tone**: 100% innovative — NOT mainstream Linear/Vercel/Stripe clones
- **Theme**: Dark mode only (current — keep it)
- **Anti-references**: Generic SaaS landing pages, typical gradient hero sections, "AI slop" aesthetics
- **Current strengths to build on**: Glassmorphism, neural/brain visual metaphors, particle animations, 3D card effects
- **Color palette** (keep): Violet (#8B5CF6), Cyan (#06B6D4), Emerald (#10B981) — these work, evolve them further
- **Direction**: Push further into the "living memory" neural aesthetic — make it feel like an organism, not a dashboard

### Design Principles

1. **Memory is Living, Not Static**
   - Visual language should feel organic, neural, interconnected — not rigid grids
   - Use floating particles, pulsing nodes, connection flows
   - Avoid boxy corporate layouts — embrace asymmetry where it serves

2. **Sovereignty Through Transparency**
   - UI should feel open, explorable, not a "black box"
   - Show the layers, show the connections, let users see their memory structure
   - Visual metaphors: unlocked doors, open vaults, visible networks

3. **Innovation Over Familiarity**
   - When faced with "standard SaaS pattern" vs "unconventional but better", choose unconventional
   - The current glassmorphism + neural effects are already differentiating — protect that
   - Don't retrofit into "safer" patterns just because they're familiar

4. **Performance as Trust**
   - Animations should be smooth (60fps), not janky
   - Reduced motion support is already there — keep it
   - Fast load times signal "this is professional-grade infrastructure"

5. **Dark Mode Excellence**
   - Dark isn't an afterthought — it's the primary experience
   - Ensure contrast ratios for accessibility within dark palette
   - Use the violet/cyan/emerald accents strategically for hierarchy
